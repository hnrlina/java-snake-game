## Key Features
* **Modern Visuals:** Uses `Graphics2D` with **Anti-aliasing** for smooth edges and professional rendering.
* **Smart Grid System:** Logic based on a 25px unit grid, ensuring the snake and apples always align perfectly.


## Built With
* **Java SE:** Core game logic and OOP structure.
* **Swing & AWT:** Graphical user interface and rendering.

### Prerequisites
* **Java Development Kit (JDK)** 8 or higher installed on your machine.
* A terminal or an IDE (VS Code, IntelliJ, or Eclipse).

## Getting Started
1. **Download** the src file
2. **Compile:** \`javac GamePanel.java\`
3. **Launch:** \`java Main\`

## How to Play
* **Arrow Keys:** Change snake direction.
* **Objective:** Eat as many apples as possible without hitting the walls or yourself!

## Project Structure
* `Snake.java`: Entry point of the application.
* `GameFrame.java`: Handles the window (JFrame) configuration.
* `GamePanel.java`: Contains the game loop, logic, and custom rendering.
